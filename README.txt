------------------------------------------------------------
HOW TO RUN USING COMMAND PROMPT (CMD)
------------------------------------------------------------

1. Open Command Prompt:
   Press Win + R → type cmd → Enter.

2. Navigate to the extracted folder:
   Example:
       cd C:\Users\<YourName>\Downloads\assignment_submission

3. To open a file in your default browser:
       start template1_problem1.html

4. To open a file in a specific browser:
       start chrome template1_problem1.html
       start msedge template1_problem1.html
       start firefox template1_problem1.html

5. To open a screenshot image:
       start screenshots\screenshot_1.png

6. (Optional) To run a local server (requires Python installed):
       python -m http.server 8000
   Then visit http://localhost:8000 in your browser to view files.
